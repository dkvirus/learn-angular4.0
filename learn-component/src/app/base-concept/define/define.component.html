1、什么是组件


组件（Component）是构成Angular应用的基础和核心。 通俗来说，组件用来包装特定的功能，应用程序的有序运行依赖于组件之间的协同工作。 以我们汽车而言：轮胎、发动机、变速箱就都是组件。细化下来就是金属零件。 我们的Angular项目就是一颗组件树！

更通俗的来说，html 页面中有很多标签，如 p 标签、table 标签、h1 标签等，这些标签都有特定的功能。
这些 html 标签针对 html 来说就可以看成是组件。html 页面就是由这些标签组成的。
Angular 也可以自定义类似 html 标签的自定义标签，如 hello-world 标签，而 Angular 页面就是由 Angular 自定义的这些标签组成，这些标签叫做 Angular 组件。

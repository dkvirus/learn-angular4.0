模板表达式定义 （代码对应目录为：app/template-expression/userage2）


定义


应用场景

禁用写法
1. 赋值 (=, +=, -=, ...)
2. new运算符
3. 使用;或,的链式表达式
4. 自增或自减操作符 (++和--)
5. 位运算符 & 或 |
6. 不能引用全局命名变量，如：window 、 document
7. 不能调用 console.log 、 Math.max 等全局函数。



可用写法
1. 管道 |
使用前需要在 .module.ts 文件引入外部模块 {{"import { CommonModule } from '@angular/common';"}}
测试： {{ today | date }}
2. 安全导航操作符 ?.
在之前的写法一般是 person && person.name ，有了安全导航操作符之后的写法变成 person?.name
测试哪怕 person 为空，person.name 只会不显示而已，并不会报错。因此作用并不大。
3. 非空断言操作符 !.
4. 逻辑运算符：&& 和 || 和 !
